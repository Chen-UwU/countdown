# 倒计时

目前能进行首考/高考的倒计时，支持调整文字，背景图片。
后续会支持更多内容。

## 🍗食用方法

### 🥳如果你是用户

下载release中的压缩包，解压放到你想放的地方。

然后给exe文件创建一个快捷方式，放入开机自启文件夹中。

就能体验到实时倒计时辣！

### 💡如果你是开发者

👉本项目支持使用poetry进行依赖安装！
```cmd
poetry install
```
如果你不知道什么是poetry，这里是[poetry官方文档](https://python-poetry.org/docs/)

👉本项目也支持使用pip进行依赖安装！
```cmd
pip install -r requirements.txt
```
