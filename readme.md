# 首考倒计时

目前能进行首考的倒计时，比如首考高考这些，现在支持调整文字。
后续会支持更多内容。

## 🍗食用方法

### 🥳如果你是用户

下载release中的压缩包，解压放到你想放的地方。

然后给exe文件创建一个快捷方式，放入开机自启文件夹中。

就能体验到实时倒计时辣！

⛔可以调整文字的版本尚未编译为exe，需要自行解压

### 💡如果你是开发者

👉本项目的依赖是：
```
pydantic
pillow
cn2an
```
